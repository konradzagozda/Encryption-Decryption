# Encryption-Decryption

## Command Line program for decrypting / encrypting messages.

### Options:
#### obligatory:
- -alg {'unicode', 'shift'}
#### Choose one from those:
- [-data {String message}]
- [-in {String filepath}]
#### arbitrary arguments:
- [-mode {'enc','dec'}]
- [-key {Integer key}]
- [-out {String filepath}]
